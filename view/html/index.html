Overview

Library API is an API that created for a school or college library that wanted to create an application that have borrow and return books functionality for their user. This API also allowed their user to do authentication features, such as login and register, so, the library can just uses the API without implementing basic features again for the application.

BASE URL for this API is

https://simple-library-api.herokuapp.com

Authentication Endpoint

Library API support several authentication functionality, such as login, register, and logout endpoint. Below is the endpoint detail for each feature.

Register

POST

Register endpoint is used to register user data, so they can login to the application. Below is the example request and response to call this endpoint. By default, every registered user will have student role. The role should be manually changed in the database become admin if necessary.

Body

Response

{

"name": "User 1",

"gender": "male",

"email": "user1@gmail.com",

"password": "user1@1234",

"dob": "2001-09-19",

}

Login

POST

Login endpoint is used to sign user in to the application. Below is the example request and response to call this endpoint.

Body

Response

{

"email": "user1@gmail.com",

"password": "user1@1234"

}

Logout

GET

Logout endpoint is used to sign user out from the application. Below is the example response for this endpoint.

Header

Response

{

"msg": "Logout success."

}

Book Endpoint

Library API support several book-related functionality, such as get books, get book by id, search book, create book, update book, and delete book endpoint. Below is the endpoint detail for each feature.

Get Books

GET

Get Books endpoint is used to retrieve all books from the database. Below is the example request and response to call this endpoint.

Query String

Response

Query String Required Default Value Description
limit False 2 Set query limit from database
page False 1 Set page for pagination purpose
sort False -createdAt Sort query by book attribute (Put - sign for descending)
